# About the project

### Tech

- Node.js (Express)
- MongoDB

### Db Info

- Users
  - One user has multiple pet
- Pets
  - One kind of pet can be fed by multiple users
### Install
```bash
cd human-and-pet
mv env .env # change your variables in .env file
yarn
yarn start
```
