# About the project

### Tech

- Node.js (Express)
- MongoDB

### Db Info

- Users
  - One user has multiple pet
- Pets
  - One kind of pet can be fed by multiple users
### Install
```bash
cd human-and-pet
echo 'PORT={your port}\nDB_NAME={your mongodb name}\nPRIVATE_KEY={your jwt secret}' >> .env
yarn
yarn start
```
